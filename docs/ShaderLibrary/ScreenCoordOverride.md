# [[ScreenCoordOverride]] - 屏幕坐标覆盖

## 概述

- **文件路径**：`Packages/com.unity.render-pipelines.core/ShaderLibrary/ScreenCoordOverride.hlsl`
- **主要职责**：提供屏幕坐标覆盖工具
- **使用场景**：特殊渲染需求

## 依赖关系

- **依赖的文件**：[[Common]] - 基础类型
- **被依赖的文件**：需要覆盖屏幕坐标的着色器

## 核心功能

允许覆盖默认的屏幕坐标计算，用于特殊渲染场景。
