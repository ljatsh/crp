# [[ShaderDebugPrint]] - 着色器调试打印

## 概述

- **文件路径**：`Packages/com.unity.render-pipelines.core/ShaderLibrary/ShaderDebugPrint.hlsl`
- **主要职责**：提供从着色器打印调试信息到控制台的函数
- **使用场景**：着色器调试

## 依赖关系

- **依赖的文件**：[[Common]] - 基础类型
- **被依赖的文件**：需要调试打印的着色器

## 核心功能

允许着色器向控制台输出调试信息，用于调试和开发。

## 与其他模块的关系

- [[Common]]：依赖基础类型
- [[Debug]]：调试工具的一部分
