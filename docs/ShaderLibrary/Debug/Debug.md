# [[Debug]] - 调试工具

## 概述

- **文件路径**：`Packages/com.unity.render-pipelines.core/ShaderLibrary/Debug.hlsl`
- **主要职责**：提供着色器调试工具函数
- **使用场景**：着色器调试和可视化

## 依赖关系

- **依赖的文件**：[[Common]] - 基础类型
- **被依赖的文件**：调试着色器

## 核心功能

提供各种调试可视化函数，如颜色编码、向量可视化等。

## 与其他模块的关系

- [[Common]]：依赖基础类型
- [[ShaderDebugPrint]]：着色器调试打印
